# Garden Near Forest

A dependency-free, playable browser prototype for the cozy magical café game.

## Play

Open `index.html` in a browser, or run a simple local web server in this folder.

## Included interactions

- Gather day/night ingredients with timed regrowth
- Plant, water, grow, weed, and harvest garden plots
- Cycle time and sunny/rainy/foggy weather
- Help Juniper the fox with a Fairy Mint quest
- Discover a memory fragment and flashback
- Raise Forest Harmony and unlock milestones
- Place garden decorations
- Open the four-area travel map
